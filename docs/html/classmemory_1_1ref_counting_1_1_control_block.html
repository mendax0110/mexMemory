Detailed Description

template<typename T, typename Allocator = DefaultAllocator<T>>
class memory::refCounting::ControlBlock< T, Allocator >

ControlBlock is a reference counting control block for managing the lifetime of objects. It supports strong and weak references, and provides thread-safe reference counting.

Template Parameters
TThe type of object being managed.
AllocatorThe allocator to use for memory management (default is DefaultAllocator).
