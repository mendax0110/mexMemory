Detailed Description

template<typename T, typename Allocator = DefaultAllocator<std::remove_extent_t<T>>>
class memory::refCounting::WeakRef< T, Allocator >

Forward declaration of WeakRef class.

WeakRef is a weak reference type that allows access to an object without affecting its lifetime.

Template Parameters
TThe type of object being referenced.
AllocatorThe allocator to use for memory management.
TThe type of object being referenced, can be a single object or an array.
AllocatorThe allocator to use for memory management, defaults to DefaultAllocator.
